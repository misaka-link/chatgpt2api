## 1.5.1 - 2026-07-09

+ [新增] 新增后台可配置图片 SSE 流硬超时，便于控制长时间挂起的图片流式请求。
+ [新增] 新增生图模型回退配置，支持主模型失败后按候选模型自动兜底重试。
+ [新增] 新增导入已导出账号 JSON、邮件 provider 边界检查与用户补全调用。
+ [新增] 新增注册机账号额度定时刷新能力，补充 YYDS 邮箱学习模式重试与按域名维护黑名单/信任列表。
+ [新增] 新增后台可配置显示时区，默认 `Asia/Shanghai`，统一覆盖日志、号池、图片管理、备份、用户密钥等页面时间显示。
+ [新增] 新增出图后删除本地对话记录选项，减少本地会话残留。
+ [优化] 合并上游图片队列解阻、推理强度调试、多方法静态资源路由等更新。
+ [优化] 调整 YYDS 学习面板布局，支持一键清空域名列表。
+ [优化] 日志管理页面支持按当前筛选条件一键清空全部日志。
+ [优化] 版本更新检测、GitHub 跳转与发布历史读取切换到 `misaka-link/chatgpt2api` 仓库。
+ [优化] Dockerfile 与默认 compose 配置补充 `TZ=Asia/Shanghai`，便于容器内日志查看。
+ [修复] 修复 sub2api 导入包装响应解析与分组过滤问题。
+ [修复] 修复前端静态资源 404/405 路由处理问题。
+ [修复] 修复图片队列前序空结果卡住问题。
+ [修复] 修复 sentinel 登录链路 `so_token` / quickjs 依赖与启动问题。
+ [修复] 修复 GitHub 镜像构建缺少 `libc6-dev` 导致的构建失败。
+ [修复] 修复 `curl_cffi` Session 资源泄漏、CloudMail token 刷新与注册邮箱 API 限流冷却问题。

## 1.5.0 - 2026-06-13

+ [新增] 新增 WARP / Privoxy / FlareSolverr 清障方案，注册遇到 Cloudflare 拦截后可刷新 clearance 并重试。
+ [新增] 新增 `outlook_token` 邮箱池，支持 Outlook/Hotmail 注册验证码读取。
+ [新增] 新增网页搜索兼容接口、第三方应用设置与无限画布入口。
+ [新增] 新增图片编辑 mask 遮罩、图片冗余倍率配置和号池 `image_inflight` 展示。
+ [新增] 新增列出正常 token 的方法，便于号池筛选和运维排查。
+ [优化] 更新 sentinel/PoW 获取方式，提高上游请求兼容性。
+ [优化] 调整代理优先级、邮件代理配置与注册请求重试逻辑。
+ [修复] 修复手动刷新时自动移除失效账号、图片 TLS 伪装默认值以及 Cloudflare 误判问题。
+ [修复] 修复 bind mount 场景下 `proxy_runtime` 配置读取异常和 Outlook 注册授权流程问题。

## 1.4.1 - 2026-06-03

+ [新增] 账号刷新改为异步模式，支持前端轮询刷新/重新登录进度。
+ [新增] 号池管理页面新增重新登录功能，支持密码登录恢复异常账号。
+ [新增] 刷新后自动重新登录异常账号（可在设置页开启）。
+ [新增] 图片生成支持并行模式，多张图片使用独立线程和账号同时生成。
+ [新增] 图片轮询超时自动换账号重试（最多4次），连接超时同账号递增等待重试。
+ [新增] 图片二次确认机制与先check再hit可配置化，关闭后可跳过等待直接返回结果。
+ [新增] 图片任务进度追踪，显示当前生成步骤（上传/预热/获取token/生成中等）。
+ [新增] 图片超时后续轮询功能，前端显示"继续等待"按钮。
+ [新增] 设置页新增图片二次确认、超时等待时间、自动重新登录等配置项。
+ [新增] 新增可编辑文件请求客户端任务 ID，兼容不支持 `crypto.randomUUID` 的环境。
+ [优化] 优化生图页面滚动加载性能，图片懒加载、会话切换滚动位置保存与恢复。
+ [优化] 提高图片会话获取成功率，减少注册 Cloudflare 干扰并增强登录重试。
+ [修复] 修复图片服务 CORS 响应头与前端 checkbox 样式问题。
